What is QR Attendance System?

QR Attendance System is a cutting-edge web application designed specifically for educational institutions to manage student attendance digitally. By leveraging QR code technology and cloud-based storage, we've created a seamless experience that benefits both educators and students.

The platform enables Class Representatives (CRs) to generate unique QR codes for each class session, which students can scan using their smartphones to instantly mark their attendance. All data is synchronized in real-time through Google Firebase, ensuring accuracy and reliability.

Why We Built This

Traditional attendance methods involve several challenges:

  • Time-Consuming: Manual roll calls waste valuable class time
  • Error-Prone: Handwritten registers are susceptible to mistakes and tampering
  • Storage Issues: Physical registers require space and deteriorate over time
  • Difficult Analysis: Extracting insights from paper records is tedious
  • Environmental Impact: Significant paper consumption

QR Attendance System addresses all these pain points with a simple, elegant solution that works on any device with an internet connection.

How It Works

For Class Representatives

  1. Register: Create an account with class details, subjects, and student list
  2. Generate QR: Select a subject and generate a unique QR code
  3. Display: Show the QR code in class or share the link
  4. Monitor: Watch real-time attendance updates (5-minute window for direct marking)
  5. Approve Requests: Review and approve late attendance requests from students
  6. Download: Export attendance reports in Excel format anytime

For Students

  1. Scan: Use your smartphone to scan the QR code
  2. Enter Roll Number: Type your roll number as registered
  3. Submit: Click submit for instant confirmation (within 5 minutes)
  4. Late Request: If late, request will be sent to CR for approval

Benefits

  • Time-Saving: Reduce attendance marking time from 5-10 minutes to under 1 minute
  • Accuracy: Eliminate errors from manual data entry
  • Eco-Friendly: Go completely paperless
  • Accessibility: Access data from anywhere, anytime
  • Security: Time-limited QR codes and late request system prevent proxy attendance
  • Cost-Effective: Free to use with no subscription fees
  • Transparent: Late attendance clearly marked with approval workflow

Security Features

  • SMS OTP Verification: Mobile number verification for password reset and account deletion
  • Time-Limited QR: 5-minute direct marking window prevents misuse
  • Late Request System: Transparent approval process for attendance after 5 minutes
  • Duplicate Prevention: Students can only mark once per session
  • Firebase Security: Database access restricted by authentication
  • SSL Encryption: All data transmission encrypted
